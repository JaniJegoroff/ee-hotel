EE Hotel Booking API tests 
==========

EE hotel booking basic acceptance API tests.

Prerequisites
==========

1) You have Ruby 2.x environment installed

2) You have `bundler` gem installed

`$ gem install bundler`

Run the tests
==========

`$ bundle exec rake`
